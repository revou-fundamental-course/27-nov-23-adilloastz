Luas Segitiga

Rumus Luas Segitiga yaitu: L = 1/2 x a x t Di mana : L = Luas a = Panjang Alas t = tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu: K = S1 + S2 + S3 Di mana : K = Keliling S = Sisi A S = Sisi B S = Sisi C

Hitung Luas Segitiga

L = 1/2 x a x t L = 1/2 x 20 x 60 L = 600

Hitung Keliling Segitiga

K = S1 + S2 + S3 K = 60 + 60 + 60 K = 180
